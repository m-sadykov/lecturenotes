# Мини-ТЗ 05: Экран списка лекций (Home)

## Цель
Показать пользователю список лекций с быстрым доступом к поиску, фильтру и записи новой лекции.

## Scope
- Вывод списка `Title/Course/Date/Duration/Status`.
- Поиск по лекциям.
- Фильтр по курсу.
- Кнопка перехода в `Recorder`.

## Артефакты
- `LecturesListViewModel`.
- Экран `LecturesList`.
- Подписка на обновления Firestore (realtime).

## Критерии приемки
- Список корректно отображает все лекции пользователя.
- Поиск работает по `localizedStandardContains`.
- Фильтр по курсу изменяет выдачу без перезапуска экрана.
