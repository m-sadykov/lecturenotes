# Мини-ТЗ 09: Экран Lecture Detail с вкладками

## Цель
Предоставить единый экран просмотра результата обработки лекции по разделам.

## Scope
- Вкладки: `Summary`, `Outline`, `Glossary`, `Flashcards`, `Quiz`, `Transcript`.
- Загрузка лекции по `lectureId`.
- Отображение пустых/ошибочных состояний.
- Базовые действия: открыть practice/quiz режим.

## Артефакты
- `LectureDetailViewModel`.
- Экран `LectureDetail` с Tab API.
- Компоненты отображения для каждого раздела.

## Критерии приемки
- Все разделы корректно читают данные из модели лекции.
- Экран устойчив к частично заполненным данным.
- Переходы в practice и quiz доступны из detail.
