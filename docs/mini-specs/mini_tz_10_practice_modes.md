# Мини-ТЗ 10: Режимы практики (Flashcards и Quiz)

## Цель
Реализовать обучающие режимы для закрепления материала лекции.

## Scope
- Flashcards: переворот карточки, действия `Know/Don't know`, прогресс.
- Quiz: multiple choice (4 варианта), подсчет score.
- Экран результата quiz.
- Сохранение локальной статистики прохождения (MVP-уровень).

## Артефакты
- `FlashcardsPracticeViewModel`.
- `QuizViewModel`.
- Экраны `FlashcardsPractice` и `Quiz`.

## Критерии приемки
- Прогресс по карточкам считается корректно.
- Score quiz считается корректно для всех вопросов.
- Финальный результат отображается и повторный запуск доступен.
