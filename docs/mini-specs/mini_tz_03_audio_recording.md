# Мини-ТЗ 03: Запись лекции

## Цель
Реализовать запись аудио лекции с управлением сессией записи и сохранением локального файла.

## Scope
- `Record/Pause/Resume/Stop`.
- Таймер записи и индикация состояния.
- Контроль лимита длительности по плану подписки.
- Сохранение файла в формате `m4a`.

## Технические параметры
- Формат: AAC / m4a.
- Частота: 44.1 kHz.
- Каналы: mono.
- Битрейт: 32–64 kbps.

## Артефакты
- `RecordingManager`.
- `RecorderViewModel`.
- Экран `Recorder`.

## Критерии приемки
- Все команды записи работают стабильно.
- Таймер не расходится с фактической длительностью.
- Полученный файл воспроизводится и доступен для upload.
